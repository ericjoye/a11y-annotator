## Detailed Description

**axe finds the problems. This one helps you hand them off.**

a11y-annotator is a Chrome extension that runs 16 accessibility checks entirely in your browser, drops color-coded pins on broken elements, lets you annotate each one, and exports a Markdown report — in ten minutes, nothing leaves your browser.

**SCAN & PIN**
Click the extension icon. 16 checks run on the active page: missing alt text, low contrast, missing labels, empty links, heading hierarchy, missing lang, missing title, small touch targets, and more. Color-coded pins (red = error, yellow = warning, blue = info) land directly on the offending elements. Your team sees exactly where the problem is.

**ANNOTATE & SHARE**
Click any pin to add a note. The side panel lists every issue grouped by type with WCAG Success Criterion references (e.g., "1.1.1 Non-text Content"). When you're done, export a Markdown report with page URL, timestamp, severity, WCAG refs, element selectors, and your annotations. Copy to clipboard or download. You can also capture a screenshot with annotations overlaid.

**WHY IT'S DIFFERENT**
- **Not a scorer.** axe and WAVE give you a number. This gives you a deliverable.
- **Not a SaaS.** No account, no server, no network calls during scanning. Works on staging, localhost, NDA pages.
- **Not $99/mo.** $4.99 one-time. No subscription.

## Key Features

- 16 automated WCAG checks (missing alt text, contrast, labels, heading hierarchy, touch targets, and more)
- Color-coded pin overlay on broken elements
- Click-to-annotate pins; notes persist for the session
- Side panel issue list grouped by severity with WCAG SC references
- Markdown export (URL, timestamp, issues, annotations, selectors)
- Screenshot capture with annotation overlays
- Keyboard shortcut: Alt+A toggles the overlay
